# user-coins-server
User Coins Service

Este microsserviço permite cadastrar usuários e consultar a quantidade de moedas associadas a cada usuário.

Foi desenvolvido utilizando Node.js e Express como parte do Projeto Integrador do curso de Desenvolvimento de Software Multiplataforma da FATEC Votorantim.

Endpoints disponíveis:

POST /users → cria um usuário
GET /users/:id/coins → consulta saldo de moedas
